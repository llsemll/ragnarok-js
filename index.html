Ragnarok.JS — เกมแนว RO single-player ในไฟล์ HTML เดียว ไม่ต้องติดตั้งอะไร

🎮 ปุ่มลัด

  • คลิกซ้าย เดิน · ตี mob · คุย NPC
  • C สถานะ / B กระเป๋า / K สกิล / Q ภารกิจ
  • 1–9 ใช้ของ / ใช้สกิล (ผูกได้)
  • S หยุดเดิน · Space ตีศัตรูใกล้สุด
  • Esc ปิด panel · Ctrl+S เซฟด้วยมือ

🌐 Deploy ขึ้นเว็บ (GitHub Pages)

ไฟล์ทั้งหมดอยู่ใน ~/Desktop/games/ragnarok/index.html ไม่ต้อง build:

  • 1. สร้าง repo ใหม่บน GitHub (เช่น ragnarok-js)
  • 2. push index.html ขึ้นไปที่ branch main
  • 3. Settings → Pages → Source: main branch / root
  • 4. รอ ~30 วิ → URL: https://USER.github.io/ragnarok-js/

หรือใช้: Netlify Drop (ลาก index.html ลง netlify.com/drop) · Vercel (vercel CLI) · Cloudflare Pages

💾 ระบบเซฟ

เซฟอัตโนมัติทุก 5 วิลง localStorage · ไม่ต้องล็อกอิน · เซฟแยกต่อ browser

🛠 แก้/ปรับเอง

เปิด index.html ใน editor — ทุกอย่างใน 1 ไฟล์ (~5,000 บรรทัด)

  • ปรับ stat: หา BASE_EXP_TABLE / recalcStats
  • เพิ่ม mob: หา MOB_TYPES
  • เพิ่มของ: หา ITEMS
  • เพิ่ม skill: หา SKILLS
  • เพิ่ม quest: หา QUESTS

Procedural sprites · Web Audio synth · No external assets · MIT License (do whatever)
